# TypeScript Pac-Man

I'll be fleshing this out in the coming weeks, but to run:

1. `git clone https://github.com/jamesseanwright/ts-pac-man.git` (or fork and use the SSH URL)
2. `cd ts-pac-man`
3. `nvm install`
4. `npm i`
5. `npm run dev`
